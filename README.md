# calculator-app

A modern and responsive calculator built using **HTML, CSS, and JavaScript**. The application features a clean Glassmorphism UI, smooth animations, keyboard support, and a fully responsive layout that works seamlessly across mobile, tablet, and desktop devices.

---

## ✨ Features

### User Interface

* Modern Glassmorphism Design
* Beautiful Gradient Background
* Responsive Layout for All Screen Sizes
* Rounded Interactive Buttons
* Smooth Hover and Click Animations
* Clean and Minimal User Experience

### Calculator Functions

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)
* Percentage (%)
* Decimal Number Support

### Controls

* AC (All Clear)
* DEL (Delete Last Character)
* Real-Time Display Updates
* Error Handling for Invalid Expressions

### Keyboard Support

* Number Keys (0–9)
* Arithmetic Operators (+, -, *, /, %)
* Decimal Point (.)
* Enter Key → Calculate Result
* Backspace Key → Delete Last Character
* Escape Key → Clear Screen

### Responsive Design

* Optimized for Mobile Devices
* Tablet-Friendly Layout
* Desktop Compatible
* Adaptive Button and Display Sizing
